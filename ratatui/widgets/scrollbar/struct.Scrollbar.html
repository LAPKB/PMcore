Expand description

Scrollbar widget for tui-rs library.

This widget can be used to display a scrollbar in a terminal user interface. The following components of the scrollbar are customizable in symbol and style.

<--▮------->
^  ^   ^   ^
│  │   │   └ end
│  │   └──── track
│  └──────── thumb
└─────────── begin
