Expand description

Implements the adaptive grid algorithm for support point expansion.

This function generates up to 2 new support points in each dimension for each existing support point. New support points are symmetrically placed around the original support point, at a distance of eps * (range_max - range_min). If the new support point is too close to an existing support point, or it is outside the given range, it is discarded.
