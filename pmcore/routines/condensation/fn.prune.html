Expand description

Prunes the theta array based on the candidate array and limits.

This function checks if the candidate support point is within the limits defined by the user and also if is not too close to the current support points.

If the candidate is within the limits and is not too close to the current support points, it is added to theta. Otherwise, it is discarded.

  • min_dist - The minimum allowed distance between the candidate and the current support points.
