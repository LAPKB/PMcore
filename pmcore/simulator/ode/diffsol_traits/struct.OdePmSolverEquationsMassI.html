fn algebraic_indices(&self) -> <Self::V as Vector>::Index

For semi-explicit DAEs (with zeros on the diagonal of the mass matrix), this function returns the indices of the algebraic state variables. This is used to determine which components of the solution vector are algebraic, and therefore must be solved for when calculating the initial condition to make sure that the algebraic constraints are satisfied. The default implementation returns an empty vector, assuming that the mass matrix is the identity matrix.
